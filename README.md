# Movimento IA — De Liso a Rico com um Claude e um Sonho

> Projeto de construção de identidade, movimento e presença no Instagram documentando a jornada de Arthur Reis: engenheiro criativo, autodidata, quebrado três vezes — reconstruindo tudo ao vivo com IA.

## Estrutura do Projeto

```
movimento-ia/
├── docs/
│   ├── MANIFESTO.md          # Identidade do movimento, crenças, manifesto
│   ├── IDENTIDADE.md         # Nome, bio, alter ego, visual identity direction
│   ├── MOVIMENTO.md          # Arquitetura Ganz: Story of Self, Us, Now
│   ├── JORNADA.md            # Log interno da jornada de criação deste projeto
│   └── ROADMAP.md            # Fases e milestones do crescimento
├── estrategia/
│   ├── pilares.md            # Os 3 pilares de conteúdo destrinchados
│   ├── metricas.md           # O que medir e quando
│   └── scamper-engine.md     # Sistema SCAMPER para geração de ângulos
├── conteudo/
│   ├── calendario/           # Planejamento semanal/mensal
│   ├── roteiros/             # Roteiros dos vídeos (por número)
│   └── publicados/           # Registro do que foi ao ar
└── assets/                   # Referências visuais, logos, paleta
```
